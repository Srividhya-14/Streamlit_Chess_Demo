# Streamlit_Chess_Demo
[Streamlit link](https://appchessdemo-87bhlghcryqer7xbafgohm.streamlit.app/)


# Commands for setup

```
python3 -m venv venv
source venv/bin/activate
pip install streamlit
pip install python_chess
streamlit run streamlit_chess_app.py
```
We can then deploy using Streamlit Community Cloud
